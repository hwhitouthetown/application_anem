Documentation administrateur, destinée aux administrateurs de l'application. Vous avez accès aux sections citées ci-dessous uniquement si vous êtes connecté en tant qu'administrateur.

Le tableau

Lorsque vous arrivez sur la partie de gestion d'utilisateurs, vous trouvez le tableau suivant :

Le tableau affiche tous les utilisateurs, pour chacun il détaille le nom, le prénom, l'agence dans laquelle il travaille. Il récapitule aussi ses droits, c'est-à-dire s'il est administrateur ou simple utilisateur de l'application. Il est aussi possible de supprimer un utilisateur, pour cela il faut cliquer sur la croix dans la dernière colonne du tableau.
Les données sont aussi modifiables, pour cela voir la catégorie suivante : Modification utilisateurs
